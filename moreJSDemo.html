Objects are a special type of variable that can contain multiple values in name:value pairs. They are used when multiple things need the same information, but also need to be separate, like two people: John Smith and Jane Doe.
  • Objects are declared or instantiated like this: var person1 = {name:"John Smith", age:25, hairColor:"blonde", eyeColor:"green"}
  • In the previous example, name, age, hairColor and eyeColor are "properties", or something that object has.
  • You can also have functions assigned to an object. An object with a method looks like this: var person1 = {name:"John Smith", age:25, hairColor:"blonde", eyeColor:"green", birthday:function() { age++; }}
  • Object functions are called methods.
