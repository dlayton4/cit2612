1. Start by opening your web console.
  2. You'll see a message: "Starting code".
  3. JavaScript code can be executed alone, or in named blocks, called functions.
    • "Starting code" was executed as soon as the page was ready, outside of any function.
  4. Three functions are provided to demonstrate basic JavaScript syntax: loopy, condish, arrayPlay, getFromArray, addToArray, and reassign.
    • loopy takes one argument, like this: loopy(5). It will count up to the number provided, then down from there to 1.
    • condish takes two arguments, like this: condish('a','b'). It compares the two values, if possible, and logs a message.
    • arrayPlay demonstrates a basic array. You can call it in the console like this: arrayPlay();
    • getFromArray returns an item from the array. Arrays start on item 0, so the first item has index 0, the second item has index 1, and so on. Try it out like this: getFromArray(1);. Hint: try getFromArray(3);
    • addToArray adds another item to the array. Try it out! addToArray("Nissan");
    • reassign takes an int (which position to reassign) and another car: reassign(1, "Lamborghini");. If you try a position that's too big, it will just put your new car at the end of the array.
    • You can try them out in the console!
