JSON stands for JavaScript Object Notation. It is an easy way to represent and transmit data contained in objects.

    For this example we have an object var car = { make: "Volkswagen", model: "Jetta TDI", year: 2016, color: "blue", engineInfo: { hp: 150, liters: 2.0, cylinder: 4 }, mpg: { city: 31, hw: 46} }
  • JSON.stringify(obj) converts a JavaScript object into a string in JSON format. Try console.log(JSON.stringify(car)) in your console.
  • JSON.parse(str) converts a string in JSON format into a JavaScript object. Try console.log(JSON.parse(str)) in your console.
